# Documentation Files

This directory contains markdown documentation files that are served through the Integration Engineer Hub.

## Files

- **training-guide.md** - NetSuite Training Guide (HTML rendering)
- **user-guide.md** - Official User Guide (markdown format)

## Usage

These files are automatically served through the web interface:

- Training Guide: `/training-guide` (rendered as HTML)
- User Guide: `/user-guide` (rendered as HTML with markdown styling)

## Adding Content

Simply replace the placeholder content in the respective `.md` files with your documentation.
